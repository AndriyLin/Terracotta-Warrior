Below is a brief summary of the samples and tests included with the Cocos2d distribution (0.99.4 beta).

== AccelViewPort test == * landscape, fps, large bg, show touch pos, anim/rotate/spin man, several figures

ActionManagerTest
  • landscape, fps, rotates/move actions
ActionsTest
  • landscape, fps, move, rotate, scale, jump, bezier, blink, fade, tint, anim, sequences

spawn (multiple simultaneous actions), delayed. repeat, orbitcamera, followaction, propertyaction

ActionsWithBlocks
  • compile fails for me :( -

Invalid value 'coop.plausible.blocks.compilers.gcc.4_2' for GCC_VERSION

To make this compile, you need to install Plausible Blocks, an SDK3.0 Blocks implementation.

AtlasTest
  • landscape/fps, demo of using textures (aka atlas?), font atlas, able to change colors/opacity/rotation/move/alignments - use as sprites
AttachTest
  • portrait,fps, figure moves to touched spot, not sure whats the 'attach' bit..
Box2dAccelTest
  • landscape, fps, box2d physics engine demo, boxes falling from sky in physical kind of way
Box2dTestBed
  • landscape, fps, many physics demos
ChipmunkAccelTouchTest
  • portrait, fps, chipmunk physics stuff
ChipmunkTestbed
  • portrait, fps, more chipmunk demos
ClickAndMove test
  • portrait fps, touch and grossini rotates to the touch point.
CocosDenshionDemo (x4)
  • no orientation, simple sound pad thing, touch button for a sound.
  • There are 4 of these test:
    • Tom the Turret
    • Drum Pad
    • Fade to Grey
    • Fancy Rat Metering
CocosLiveTest
  • demo of the high score facility.
CocosNodeTest
  • landscape, fps, camera, rotations, anchorpoints, children, tags, remove/cleanup +with children, stress tests, nodeToParent, camera orbits/zoom
DrawPrimitivesTest
  • fps, draws primitive shapes/curves etc, next/prev re-orients iphone.
EAGLView Test
  • Creates an EAGLView using Interface Builder
EaseActionsTest
  • landscape, fps, ease in/out, exp in/out, etc, elastic in/out, bounce in/out, scheduler, scaleTime
Effects Test
  • landscape, fps, Shaky3D, Waves3D, FlipX3D, FlipY3D, Lens3D, Ripple3D, Liquid3D, Waves, Twirl,

ShakyTiles3D and many more…

Effects Advanced Test
  • landscape, fps, combining 2 or more effects at once - funky.
Font Test
  • Tests and Shows how to use custom TTF fonts
Interval Test
  • Tests the CCNode scheduler API
Layer Test
  • landscape, fps, couple of color/blend layer tests
Menu Test
  • landscape, fps, test different menu/button styles
Menu Test with Blocks
  • Tests MenuItems with Blocks. It needs PLblocks.
Motion Streak Test
  • landscape, fps, examples of how you add motion streaks to your sprites.
Parallax Test
  • landscape, fps, examples using parallax, auto move and touch to move.

== Particle View == * portrait, fps, shows fireworks type effects

Particle Test
  • portrait, fps, wow - lots of fireworks type effects
PASoundEngineTest
  • stereo sound tests?
PerformanceTestChildrenNode
  • Landscape, fps, does background testing - see console.
  • Need to enable profilers for it to compile, ie, in ccConfig.h, change this:

#define CC_ENABLE_PROFILERS 1

PerformanceTestParticles
  • portrait, fps, vary speeds/sizes
PerformanceTestSprites
  • landscape, fps, lots of sprites, some rotating
PerformanceTestTouches
  • landscape, fps, touches - targeted and standard (huh?)
ProgressActionsTests
  • landscape, fps, ProgressTo, ways to make sprite appear.
RenderTextureTest
  • landscape, fps, like a little drawing app.
RotateWorldTest
  • landscape, fps, lots of rotations
SceneTest
  • landscape, fps, samples switching scenes.
SchedulerTest
  • landscape, fps, tho its not a visual test, lots of tests, via console
SpriteTest
  • landscape, fps, lots of sprites, animated
Texture2dTest
  • landscape, fps, different texture tests, anti-alias, PVR, mipmap, rotates, other image types, blending, GL_CLAMP, GL_REPEAT
TileMapTest
  • landscape, fps, tile options, ortho, hex, Iso, Zorder
TouchesTest
  • portrait, fps, ping pong game like test
TransitionsTest
  • landscape, fps, CCJumpZoom, CCCrossFade, CCRadialCCW, CCRadialCW, PageTransitionForward/Backward, CCFadeTR/BL/Up/DownTransition, CCTurnOffTiles, CCSplitRows/Cols, CCFade, FadeWhite, FlipXLeft/RightOver, FlipYUp/DownOver, FlipAngularLeft/RightOver, and lots more cool transitions…
HelloWorldSample
  • landscape, just says Hello World :)
HelloActionsSample
  • landscape, moving text and sprite, via actions
HelloEventsSample
  • landscape, text and sprite, sprite moves to touched position.
Bug422/Bug350
  • bug specific test routines.
More to come...
  • link to source/main part of each test - in source repo.
